## config

### set
```
git config --global user.name username
git config --global user.email test@gmail.com
```

### get
```
git config user.name
```

## branch
### show all branches
```
git branch -a
```

### show remote branches
```
git branch -r
```

### delete branch
```
git branch -D branch_name
```

### switch branch
```
git switch branch_name
```
